# Task : Refactor Vehicle Classes for Liskov Substitution Principle
## Description : 
Create a Vehicle base class with 'startEngine' and 'stopEngine' methods. Implement subclasses such as 'Car' and 'ElectricCar'. Ensure that replacing instances of the base class with subclass instances works seamlessly.
## Topics Covered : 
Liskov Substitution Principle
## Objective : 
Learner will understand how to design classes and subclasses that follow the Liskov Substitution Principle.
