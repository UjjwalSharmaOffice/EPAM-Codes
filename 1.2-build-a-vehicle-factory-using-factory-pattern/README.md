# Task : Build a Vehicle Factory using Factory Pattern
## Description : 
Design a factory class that creates different types of vehicles such as 'Car', 'Bike', and 'Truck'. Use the Factory pattern to encapsulate the creation logic.
## Topics Covered : 
Creational Design Patterns, Factory
## Objective : 
Learner will learn how to use the Factory pattern to simplify object creation and promote maintainable code.
